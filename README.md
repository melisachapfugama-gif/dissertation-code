# dissertation-code

This repository presents an end-to-end machine learning framework designed to assess the shifting landscape of web scraping ethics and digital data governance by leveraging a multi-model topic modeling and clustering workflow for systematic literature reviews. Built using Python and core computational linguistics libraries such as Tomotopy, Scikit-Learn, Gensim, and Sentence-Transformers, the system features a robust preprocessing pipeline that handles deep text cleaning, tokenization, and customized stop-word filtering across extracted scholarly corpora. To capture both structural patterns and deep semantic nuances, the architecture integrates a diverse feature engineering matrix, spanning traditional Bag-of-Words (BoW) vectorization, high-dimensional TF-IDF matrices, and dense 384-dimensional contextual neural embeddings (BERT).

Furthermore, the repository implements systematic hyperparameter tuning loops to simultaneously evaluate U_Mass coherence, model perplexity, and geometric silhouette scores across varying cluster ceilings ($k$), enabling automated optimal theme selection and comprehensive diagrammatic keyword extraction. Uniquely, the framework couples these structural models with an Ordinary Least Squares (OLS) regression engine to track temporal covariate shifts, reporting absolute $p$-values to verify statistically significant thematic trends over time. Ultimately, this benchmarking tool accelerates data science and governance research by turning massive, unstructured text datasets into structured, mathematically validated, and policy-relevant insights.
